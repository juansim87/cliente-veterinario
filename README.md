## Descripción del Proyecto
Proyecto de React, Vite y Tailwind dirigido a tomar un primer contacto con estas tecnologías. El proyecto consta de un formulario para introducir los datos del cliente, su mascota y 
las afecciones de la misma. Se generan entradas dinámicas que se pueden editar o eliminar.

---

## Objetivos
1. Tener una primera toma de contacto con Vite y Tailwind.
2. Crear un gestor sencillo de anotación de citas, con un formulario y una lista creada dinámicamente en la que se puedan editar los datos o eliminar la entrada.
3. Hacer el proyecto responsivo para que se pueda acceder desde móvil, tablet, portátil o sobremesa.
